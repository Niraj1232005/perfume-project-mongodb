# Perfume Shop

A full-stack perfume shop web application built using React, Node.js, and MongoDB.

## Features
- Responsive homepage with product listings
- Product detail page with image gallery
- Review system (add & view reviews)
- Hover image effect on product cards

## Tech Stack
- Frontend: React (Vite)
- Backend: Node.js, Express
- Database: MongoDB Atlas

## How to Run

### Backend
```
cd server
npm install
node server.js
```
### Frontend
```
cd client
npm install
npm run dev
```

## Note
- Make sure MongoDB Atlas connection is active
- Backend runs on http://localhost:5000
- Frontend runs on http://localhost:5173
